## المميزات

### 🕐 أوقات الصلاة (تلقائي حسب الموقع)
- تحديد الموقع تلقائياً عبر GPS المتصفح
- أوقات الصلوات الخمس (الفجر، الظهر، العصر، المغرب، العشاء)
- عرض الصلاة القادمة والوقت المتبقي
- التاريخ الهجري والميلادي
- يعمل مع أي موقع في العالم (API: aladhan.com)

### 📖 القرآن الكريم
- ١١٤ سورة كاملة بالخط العثماني
- بحث في السور بالعربية والإنجليزية
- عرض الآيات بخط أميري القرآني
- تحميل مباشر من API (alquran.cloud)

### 🤲 أذكار الصباح والمساء
- أذكار الصباح (١٠ أذكار مع العدد الصحيح)
- أذكار المساء
- أذكار النوم (تسبيح فاطمة)
- عدّاد تفاعلي لكل ذكر مع تحديد الانتهاء

### 📿 عدّاد التسبيح
- عدّاد رقمي بتصميم جميل
- ٦ تسبيحات: سبحان الله، الحمد لله، الله أكبر، لا إله إلا الله، أستغفر الله، لا حول ولا قوة إلا بالله
- اهتزاز عند كل ضغطة
- إجمالي تراكمي

### 🕌 الأدعية المأثورة
- ١٠ أدعية من القرآن والسنة
- دعاء ربنا آتنا، دعاء يونس، دعاء الهم والحزن
- المصدر لكل دعاء

### ☪️ السنن اليومية
- ١٠ سنن يومية مرتبة بالوقت
- السواك، صلاة الضحى، السنن الرواتب، قيام الليل
- الحديث المرجعي لكل سنة

### 🧭 اتجاه القبلة
- حساب تلقائي من موقعك
- بوصلة بصرية مع سهم الاتجاه
- الدرجة الدقيقة

### 📜 الأربعون النووية
- ٧ أحاديث نووية (يمكن التوسعة)
- نص الحديث والراوي والمصدر

### ✨ أسماء الله الحسنى
- ٩٩ اسماً في شبكة جميلة

### 📻 إذاعة القرآن الكريم
- إذاعات مباشرة للقرآن الكريم والسنة

### 🌟 مميزات إضافية
- آية اليوم وحديث اليوم
- تصميم داكن أنيق بألوان إسلامية (أخضر وذهبي)
- متجاوب مع جميع الشاشات
- يعمل بدون إنترنت بعد التحميل الأول (ما عدا القرآن والصلاة)
- ملف HTML واحد — بدون خادم
