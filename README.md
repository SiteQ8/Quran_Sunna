## المميزات | Features

### الوضع الفاتح والداكن 🎨
- وضع فاتح (Light Mode) بتصميم عصري أنيق
- وضع داكن (Dark Mode) بألوان ذهبية إسلامية
- حفظ تلقائي لاختيار المظهر
- زر تبديل سريع في الزاوية العلوية

### ساعة الأذان الدائرية ⏰
- عرض دائري متحرك لوقت الصلاة
- مؤشر تقدم الوقت بين الصلوات
- عدّاد الوقت المنقضي منذ الأذان
- تحديث كل ثانية

### أوقات الصلاة — الكويت 🕌
- أوقات الصلاة الستة (الفجر، الشروق، الظهر، العصر، المغرب، العشاء)
- محسوبة لمدينة الكويت وفق طريقة أم القرى
- التاريخ الهجري والميلادي
- الصلاة القادمة مع العد التنازلي
- تحديث تلقائي كل ٥ دقائق

### القرآن الكريم 📖
- ١١٤ سورة كاملة — ٦,٢٣٦ آية
- الخط العثماني (مصحف الملك فهد) عبر AlQuran Cloud API
- الترجمة الإنجليزية (Sahih International)
- البحث في أسماء السور
- البسملة (مخفية لسورة التوبة والفاتحة)
- تكبير وتصغير حجم الخط
- عرض نوع السورة (مكية / مدنية) وعدد الآيات

### الأذكار والأدعية 🤲
- **أذكار الصباح** — ٦ أذكار مع المصدر وعدد التكرار
- **أذكار المساء** — ٤ أذكار
- **أذكار بعد الصلاة** — ٤ أذكار (منها آية الكرسي)
- **أذكار النوم** — ٣ أذكار
- **أدعية مأثورة** — ٦ أدعية من القرآن والسنة
- المصدر لكل ذكر (البخاري، مسلم، الترمذي، حصن المسلم)

### عدّاد التسبيح 📿
- ٦ أذكار للتسبيح: سبحان الله، الحمد لله، الله أكبر، لا إله إلا الله، الحوقلة، الاستغفار
- أهداف قابلة للتغيير: ٣٣، ١٠٠، ٥٠٠
- إجمالي كلّي
- اهتزاز عند الضغط (الهواتف الذكية)
- اهتزاز مميّز عند بلوغ الهدف

## المصادر | Sources

| المصدر | الوصف |
|--------|-------|
| **AlQuran Cloud API** | القرآن بالخط العثماني (`quran-uthmani`) + Sahih International (`en.sahih`) |
| **Aladhan API** | أوقات الصلاة لدولة الكويت (طريقة أم القرى) |
| **حصن المسلم** | الأذكار والأدعية — الإمام سعيد بن وهف القحطاني |
| **صحيح البخاري** | الأحاديث النبوية الصحيحة |
| **صحيح مسلم** | الأحاديث النبوية الصحيحة |

## التصميم | Design

- مخطوطة إسلامية: أزرق داكن (`#0c1a2a`) + ذهبي (`#c9a84c`)
- خط أميري قرآن للنص القرآني
- خط Noto Naskh Arabic للأذكار
- نمط هندسي إسلامي في الخلفية
- شريط تنقل سفلي (٥ أقسام)
- تصميم للهواتف أولاً (Mobile-First)
- متوافق مع PWA (إضافة للشاشة الرئيسية)
