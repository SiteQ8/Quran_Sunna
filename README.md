## المميزات

### 🕌 أوقات الصلاة
- تحديد تلقائي للموقع عبر GPS
- أوقات صلاة دقيقة (Aladhan API — طريقة أم القرى)
- ساعة دائرية مع العد التنازلي للصلاة القادمة
- التاريخ الهجري والميلادي · آية اليوم

### 📖 القرآن الكريم
- ١١٤ سورة كاملة مع بحث سريع
- خط Amiri Quran · بيانات مباشرة من alquran.cloud
- ترقيم الآيات · بسملة · معلومات السورة

### 📿 الأذكار (٢٠٠+ ذكر)
أذكار الصباح · المساء · النوم · الاستيقاظ · بعد الصلاة · أدعية قرآنية · أسماء الله الحسنى (٩٩) · أذكار متنوعة — عداد تنازلي مع اهتزاز

### ☀️ السنن اليومية
