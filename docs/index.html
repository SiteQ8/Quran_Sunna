اللَّهُمَّ ارْحَمْنِي بالقُرْآنِ وَاجْعَلهُ لِي إِمَاماً وَنُوراً وَهُدًى وَرَحْمَةً

اللَّهُمَّ ذَكِّرْنِي مِنْهُ مَا نَسِيتُ وَعَلِّمْنِي مِنْهُ مَا جَهِلْتُ وَارْزُقْنِي تِلاَوَتَهُ آنَاءَ اللَّيْلِ وَأَطْرَافَ النَّهَارِ

اللَّهُمَّ اجْعَلْهُ حُجَّةً لِي وَلاَ تَجْعَلْهُ حُجَّةً عَلَيَّ

اللَّهُمَّ اجْعَلْهُ لِي شَافِعاً وَمُشَفَّعاً يَوْمَ القِيَامَةِ

اللَّهُمَّ زَيِّنِّي بِالقُرْآنِ وَأَكْرِمْنِي بِهِ

سُبْحَانَكَ اللَّهُمَّ وَبِحَمْدِكَ، أَشْهَدُ أَنْ لاَ إِلَهَ إِلاَّ أَنْتَ، أَسْتَغْفِرُكَ وَأَتُوبُ إِلَيْكَ
